## 🔬 Overview

The **Classroom Behavior Analysis System** is an advanced AI-powered platform that uses **computer vision and deep learning** to analyze student behavior in real-time, providing educators with actionable insights about classroom engagement, attention levels, and disruptive behaviors — without intruding on student privacy.

> 💡 **Real impact:** Helps teachers improve learning outcomes by providing data-driven engagement insights from classroom camera feeds.

---

## ✨ Key Features

| 👁️ Behavior Detection | 📊 Analytics Dashboard |
|---|---|
| 🌸 Real-time student attention tracking | 📈 Real-time engagement scores |
| 📴 Sleep / distraction detection | 📉 Class attention heatmaps |
| 📰 Hand-raise and participation recognition | 📊 Historical trends & reports |
| 📵 Mobile phone / distraction detection | 🔔 Automated alerts to educators |
| 🙉 Posture and engagement scoring | 🔕 Student-level insights (privacy-safe) |

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Computer Vision** | YOLOv8, MediaPipe, OpenCV |
| **Deep Learning** | TensorFlow, PyTorch, Custom CNN |
| **Backend** | FastAPI, Python |
| **Frontend** | React, Real-time websockets |
| **Database** | PostgreSQL, Timeseries DB |
| **Deployment** | Docker, Raspberry Pi / Edge cameras |

## 🎬 How It Works

```
📷 Camera Feed → 🖼️ Frame Extraction → 🤖 YOLO Pose Detection → 🧠 Behavior Classifier → 📊 Engagement Score → 🔔 Teacher Alert
```
